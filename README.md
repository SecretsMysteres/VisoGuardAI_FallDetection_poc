
# FallGuard v2.0 – Détection de chutes (PoC open-source)

Détection robuste de chutes humaines en environnement réel (occlusions, éclairage variable)  
via **YOLOv8 segmentation + pose estimation + fusion DL**.

## Pour qui ?
- **EHPAD** & maisons de retraite  
- **Hôpitaux** & cliniques  
- **Domicile** (personnes âgées, handicap)  
- **Industrie** (sécurité travailleurs isolés)  
- **Assureurs** (prévention risques)

## Fonctionnalités
- Détection en temps réel (<40 ms sur Jetson Nano)  
- Démo live (upload vidéo → alerte)  
- 100 % open-source & reproductible
